This repository is a submission for a course project on Reproducible Research 

Data: activity data of an indiviual wearing an activity monitoring device which records the number of steps in 5-minutes intervals. Collected in October and November 2012.

Source: https://d396qusza40orc.cloudfront.net/repdata%2Fdata%2Factivity.zip

Aim: Implement a strategy for filling in the missing values in the dataset. Use the data to study activity patterns.

Done in April 2021
